# Smart Proxy Manager

Программа для удобного управления HTTP прокси в Windows (поддерживает работу в глобальном режиме и выборочное проксирование конкретных приложений). Работает на базе движка `mihomo` (Clash Meta).

## 🚀 Установка

Поскольку движок `mihomo.exe` весит достаточно много, он не включен в этот репозиторий по умолчанию. 

Для работы исходного кода вам нужно:

1. Скачать актуальную версию ядра **mihomo** (бывший Clash Meta) с их официального GitHub репозитория:
   🔗 [Релизы MetaCubeX/mihomo](https://github.com/MetaCubeX/mihomo/releases)
   *(Вам нужен файл вида `mihomo-windows-amd64-...zip` или `.exe`)*

2. В папке с проектом (где лежит `main.py`) создать папку с названием `bin`.

3. Переименовать скачанный файл ядра в `mihomo.exe` и положить его внутрь папки `bin`.

Структура должна получиться такой:
```
proxy-app/
├── bin/
│   └── mihomo.exe    <-- Скачанное ядро
├── main.py
└── ...
```

## 🛠 Сборка в EXE

Если вы скачали исходники и хотите скомпилировать приложение в один `.exe` файл (чтобы внутри него сразу был зашит `mihomo.exe`), выполните следующие шаги:

1. Установите зависимости:
   ```cmd
   pip install customtkinter pyperclip requests pyyaml
   pip install pyinstaller
   ```
2. Запустите сборку:
   ```cmd
   pyinstaller --clean main.spec
   ```
3. Готовый файл появится в папке `dist/main.exe`.

## ⚙️ Особенности
* Автоматическое закрытие повисших фоновых процессов при запуске.
* Корректное восстановление системных маршрутов (TUN) при выходе (без обрыва интернета).
* Исправленный буфер обмена `Ctrl+C` / `Ctrl+V` при работе от имени Администратора.
